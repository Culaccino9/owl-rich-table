# owl-rich-table

## Install

Using npm:

```bash
$ npm install --save owl-rich-table
```

or using yarn:

```bash
$ yarn add owl-rich-table
```

## demo
- 支持curd
- 支持宽度拉伸
- 支持高度自适应
